## Cài Đặt

* Chương trình chạy trên **python 3**.
* Cần package:
    * **requests**,
    * **bs4**
    * **PIL**
    * **rich**
    * **climage**
* Nếu chưa có package thì có thể tải package như sau:
```
python3 -m pip install -r requirements.txt
```

* Chỉ cần chạy file `main.py` là xong.
